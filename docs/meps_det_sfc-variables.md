# MEPS `meps_det_sfc` variable reference

Every variable in MET Norway's MEPS **deterministic surface** file
(`meps_det_sfc`), the source this app fetches from. Generated by enumerating
the live OPeNDAP dataset (not hand-maintained) — regenerate with the script
in this repo's history if MET changes the product.

- **Source run enumerated**: `2026-07-08T06Z`
  (`https://thredds.met.no/thredds/dodsC/.../meps_det_sfc_YYYYMMDDTHHZ.ncml`)
- **Grid**: 1069 × 949 (y × x), native Lambert (see `config.NATIVE_PROJ4`);
  **67** hourly forecast steps.
- **Total variables**: 221
  (97 forecast fields, 98 SURFEX tile
  diagnostics `SFX_*`, 26 coordinate/metadata).
- Fill value for data fields is `9.96921e36` (masked → NaN on fetch).
- `precipitation_type` category codes (0–7): 0 drizzle, 1 rain, 2 sleet,
  3 snow, 4 freezing drizzle, 5 freezing rain, 6 graupel, 7 hail.
